# 160 Bytes — 5 × 32 Chunk Structuur

## Basis
- **Totaal:** 160 bytes
- **Chunks:** 5 × 32 bytes
- **Hex:** 0xA0
- **Chunk hex:** 0x20 per slot

## Patroon
```
Slot 0: [0x0000 - 0x001F]  32 bytes
Slot 1: [0x0020 - 0x003F]  32 bytes
Slot 2: [0x0040 - 0x005F]  32 bytes
Slot 3: [0x0060 - 0x007F]  32 bytes
Slot 4: [0x0080 - 0x009F]  32 bytes
```

## Betekenis
- 5 = prime, 5th positie in 3-6-9 cyclus
- Korte cyclus voor snelle signalen
- Past in NPR 3-6-9: 5 slots → 5th resonantie
- Hexa-relevant: 5 is de halve phase + 1

## Relatie tot NPR OS
- Sub-fase van 1024 (160 × 6.4 = 1024)
- Past in 0x20 chunk-model
- Complementair aan PHASE_SIZE 0x10 (dubbel daarop)

## Digital Root
- 160 → 1+6+0 = 7
- 5 × 32 → 5 × 5 = 25 → 7
- Consistent: 7 = terugkeer naar bron
