# Verklaring van de Rechten van de Mens en van de Burger — 1795

**Datum:** 31 oktober 1795
**Type:** Rechtenverklaring

## Betekenis
Na de Bataafse Revolutie — eerste formele rechtenverklaring.

```
eenheidsstaat
+ volkssoevereiniteit
+ burgerrechten
+ machtenscheiding
+ nationale vertegenwoordiging
→ Staatsregeling 1798
```

## Beginselen
- `popular_sovereignty` — volkssoevereiniteit
- `fundamental_rights` — burgerrechten
- `equality_before_law` — gelijkheid voor de wet
- `individual_liberty` — individuele vrijheid
