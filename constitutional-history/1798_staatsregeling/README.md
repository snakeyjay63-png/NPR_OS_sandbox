## Historische Context
```
vóór 1579: graafschappen, hertogdommen, heerlijkheden
1579–1795: zeven soevereine gewesten (Republiek)
1798: gewestelijke soevereiniteit afgeschaft → eenheidsstaat met departementen
1814: "provinciën of landschappen" keert terug
1815: provincies formeel onderdeel van Koninkrijk der Nederlanden
```

De 1798 staatsregeling markeert de breuk: van bond van gewesten → eenheidsstaat.
Departementen vervangen provincies. Centraal gezag. Vertegenwoordigende macht.
