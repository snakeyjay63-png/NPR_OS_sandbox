# Constitutioneel Historisch Corpus

Nederlandse constitutionele en juridische bronnen — geordend als NPR-OS taalvelden.

## Kerninzicht

```
Grondwet       = politieke samenvoeging van rechtsvelden
waterrecht     = functionele samenvoeging door gedeeld fysiek systeem
waterdefensie  = bewuste verplaatsing van de grens land ↔ water
kwelder        = levende overgang zee ↔ land (natuurlijke voorverdediging)
```

Vier samen:

```
Noise   → gedeeld probleem (water, veiligheid, godsdienst)
Pattern → gezamenlijke regels, bestuur, controle
Return  → duurzame institutionele structuur
Buffer  → levende grens (kwelder, dijk, polder)
```
