# Waterwoordenschat — Taal Als Kaart van het Watersysteem

*gegenereerd 2026-07-18 | bron: etymologisch + functioneel overzicht*

**Kerninzicht:** De Nederlandse woordenschat is zelf een kaart van het watersysteem.
Elk woord codeert: oorsprong + richting + schaal + bodem + getij + menselijk beheer + regionale taal.

---

## Routes Door Het Poldersysteem

```
perceelsloot     → verzamelt lokaal water
tochtsloot / tocht → verzamelt water uit meerdere sloten
vliet / wetering  → voert het verder
boezem           → tijdelijk gemeenschappelijk opslag- en afvoerstelsel
boezemkanaal     → verbindt opslaggebieden en uitlaatpunten
uitwatering      → route naar rivier, meer of zee
spui             → afvoer door natuurlijk peilverschil
gemaal           → mechanische verplaatsing naar hoger water
```

**Boezem** is bijzonder: niet één losse waterloop, maar een samenhangend stelsel waarin polderwater tijdelijk wordt opgevangen voordat het verder wordt afgevoerd.

## Eén Nederlandse Waterroute

**Natuurlijk:**
```
regen → infiltratie → zijging → kwel of bron → beek → rivier → geul → estuarium → wad → priel → zee
```

**Gemaakt landschap:**
```
regen → greppel → sloot → tocht → wetering → boezem → gemaal of spui → rivier/zee
```

## NPR-Lezing Van De Woordenschat

Elk woord draagt meervoudige informatie:

```
oorsprong    (natuurlijk / gemaakt / ondergronds / getij)
+ richting   (omhoog / afwaarts / door / terug)
+ schaal     (lokaal / regionaal / nationaal)
+ bodem      (zand / klei / veen / slik)
+ getij      (getijbeïnvloed / vrij van getij)
+ menselijk beheer  (ongereguleerd / gegraven / bedamd / gepompt)
+ regionale taal    (Fries / Zeewoods / Brabants / etc.)
```

Taal = kaart. Woord = coördinaat in het watersysteem.
