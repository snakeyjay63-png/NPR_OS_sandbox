# Angkor Wat — Geneste Verhoudingen

## Drie Lagersonderscheiding

```
1. werkelijk gemeten architectuur
2. werkelijk gereconstrueerde waterroutes
3. symbolische interpretatie van die vormen
```

Niet bij losse "heilige getallen" beginnen. Bij de route:

```
landschap
→ buitenste watergrens
→ ommuring
→ verhoogde toegangsas
→ opeenvolgende omheiningen
→ centrale torengroep
```

---

## 1. Buitenframe

Gracht omheen ≈ rechthoek:

```
1.500 m × 1.300 m
```

Grachtbreedte ≈ `190 m`. Binnen de gracht, ommuurde ruimte ≈:

```
1.025 m × 800 m  (≈ 82 hectare)
```

Heel complex inclusief gracht ≈ `200 hectare`.

Eerste verhoudingen:

```
buitenframe:    1500 : 1300  =  15 : 13  ≈ 1,154 : 1
binnenruimte:   1025 : 800   =  41 : 32  ≈ 1,281 : 1
```

Geen perfecte vierkanten. **Langgerekte rechthoek met dominante west-oostas.**

```
WEST → toegang → causeway → tempelcentrum → OOST
```

---

## 2. Water als Buitenste Nulveld

```
LAND → GRACHT → EILAND/OMMUURD VELD
```

Tempel = begrensde landmassa binnen water:

```
water [ land met tempel ] water
```

NPR-notatie:

```
SOURCE_0 := watergrens
FRAME_1  := ommuurde rechthoek
CONTENT  := tempelroute
CENTER   := centrale toren
```

Gracht maakt één operatie zichtbaar:

```
continu landschap
→ onderbreking door water
→ afzonderlijk geordend veld
```

Toegang alleen via gecontroleerde oversteek. Water maakt centrum **topologisch afzonderlijk**, fysiek in hetzelfde landschap.

Symbolisch: kosmische oceaan rond wereldberg Meru.
Architectonisch: begrenzing, waterlichaam, stabiliserende landschapsstructuur.

---

## 3. Toegangsroute als Vector

Westelijke causeway → rechthoek wordt gerichte structuur:

```
watergrens → poort → lange horizontale as
→ galerijen → stijgende terrassen → centrum
```

Zonder route: concentrische vormen.
Met route: programma.

```
x₀ = buitenwereld
x₁ = water oversteken
x₂ = ommuring passeren
x₃ = open veld doorkruisen
x₄ = galerij betreden
x₅ = niveau stijgen
x₆ = centrum bereiken
```

Angkor Wat = niet alleen "rechthoek binnen rechthoek" maar:

```
rechthoek + richting + drempels + hoogteverschil
```

Iedere drempel = toestandsverandering:

```
S₀ → S₁ → S₂ → S₃ → S₄
```

Elke volgende toestand:
- kleiner in oppervlakte
- hoger
- minder toegankelijk
- dichter bij centrale as

---

## 4. Geneste Frames

```
FRAME₀
└── FRAME₁
    └── FRAME₂
        └── FRAME₃
            └── CENTER
```

Ruimtelijk:

```
gracht → buitenmuur → buitenhof → eerste galerij
→ tweede niveau → bovenste niveau → centrale toren
```

Elke overgang:

```
oppervlak neemt af    A₀ > A₁ > A₂ > A₃
hoogte neemt toe      h₀ < h₁ < h₂ < h₃
```

Inverse verhouding:

```
meer hoogte ↔ minder horizontale ruimte
```

Centrum: kleinste horizontale omvang, grootste verticale en symbolische concentratie.

---

## 5. Vijf Torens — Centrum Plus Vier Hoeken

Quincunx-configuratie:

```
  ●   ●
    ●
  ●   ●
```

```
4 buitenpunten + 1 centrum = 5 torens
5 = 4 + 1
```

NPR-taal:

```
4 = frame
1 = source/centrum
5 = centrum in viervoudige ruimte
CENTER_1 binnen CARDINAL_FRAME_4
```

Geometrisch:

```
noordwest ───── noordoost
  \           /
   \ centrum /
  /           \
zuidwest ───── zuidoost
```

Centrum ruimtelijk vastgelegd door vier hoekposities.

---

## 6. Drie Verticale Niveaus

```
LEVEL_1 → LEVEL_2 → LEVEL_3
```

```
3 niveaus × 5-torenconfiguratie (bovenin)
```

`3` = verticale lens:

```
basis → overgang → top
POINT_PRIME_3 := laag → midden → centrum
```

Drie gelijktijdige assen:

```
X := richting    (west → oost)
Y := symmetrie
Z := hoogte      (laag → hoog)
```

Plus concentrisch: buiten → binnen.

Letterlijk een **driedimensionale route door geneste frames**.

## 7. Waterverhouding Op Tempelschaal

```
gracht:  ≈ 190 m breed
totaal:  ≈ 1300 m noord-zuid
```

```
190 / 1300 ≈ 0,146    (één zijde ≈ 14,6%)
2 × 190 = 380 m
380 / 1300 ≈ 0,292    (beide zijden ≈ 29%)
```

Schematisch:

```
190 water + circa 920 land + 190 water ≈ 1300 m
```

```
WATER → LAND → WATER
0 → 1 → 0
```

Landveld = begrensde toestand binnen water.

## 8. Schalen En Systeemgrenzen

Angkor kent vier schaalniveaus:

```
Angkor Wat-tempelcomplex       ≈ 2 km² (incl. gracht)
UNESCO Archeologisch Park      ≈ 400 km²
Greater Angkor kernland        ≈ 1.000–1.500 km²
Ruimste onderzochte zone        ≈ 3.000 km²
```

Nederland tot vergelijking:

```
Nederland totaal               ≈ 41.543 km²
Nederland land                 ≈ 33.626 km²
```

Verhoudingen tot Nederland:

```
400 / 41.543  ≈ 1%      (UNESCO-park ≈ 1/100 Nederland)
1.500 / 41.543 ≈ 3,6%   (Greater Angkor kern)
3.000 / 41.543 ≈ 7,2%   (ruimste zone ≈ 1/14 Nederland)
```

Angkor Wat is **niet** zo groot als Nederland. Zelfs de ruimste zone is ≈ één veertiende.

Maar de echte overeenkomst is niet oppervlak — het is **schaalrelatie**.

## 9. Angkor Wat In Het Grotere Waternetwerk

```
Phnom Kulen / hoger terrein
→ rivieren en kanalen
→ barays en lokale bassins
→ stedelijke en agrarische velden
→ afvoer richting lager land en Tonlé Sap
```

Niet: `reservoir → irrigatie`
Maar: **netwerk** van natuurlijke rivieren, barays, kanalen, watergangen, vijvers, tempelgrachten, dijken, verhoogde wegen.

---

## 10. Schaalrelatie — De Echte Overeenkomst

Angkor Wat is slechts het centrale punt van een genest systeem:

```
TEMPEL_1
→ stad en reservoirs
→ lokaal kanalenraster
→ Greater Angkor
→ Kulen–Tonlé Sap-waterroute
```

Nederland:

```
gebouw/stad
→ polder
→ boezemgebied
→ waterschap
→ Rijn–Maas–Scheldedelta
```

Vergelijking:

```
Angkor Wat : Greater Angkor
≈ lokaal centrum : regionaal waterlandschap

Nederlandse stad/polder : Nederland
≈ lokaal centrum : nationale delta
```

Niet gelijk in grootte — gelijk in geneste opbouw.

## 11. Vergelijking Met Nederland

Niet: `Angkor Wat = Nederlandse polder`
Maar: **beide gebruiken geneste waterpeilen en begrensde velden**

### Angkor
```
hoger brongebied → kanaal → reservoir → lokaal bassin/gracht
→ veld/nederzetting → lagere afvoer
```

### Nederland
```
rivier/regen/kwel → sloot → tocht/wetering → boezem
→ gemaal/spui → rivier/zee
```

### Gedeelde Structuur
```
SOURCE → CAPTURE → STORAGE → DISTRIBUTION → USE → RETURN
```

### Verschuif
```
Angkor:     zwaartekracht + moesson → spreiden + tijdelijk bewaren
Nederland:  zee + rivier + regen → weren + bewaren + kunstmatig omhoog pompen
```

Angkor: zeer kleine hoogteverschillen, zwaartekracht.
Nederland: later molens en gemalen, tegen natuurlijk verloop in.

## 12. Genormaliseerd Coördinatenmodel

```
totale lengte   = 1500 m := 1.00
totale breedte  = 1300 m := 1.00
```

Elke positie = verhouding tussen `0` en `1`:

```
x' = x / 1500
y' = y / 1300
z' = z / max_hoogte
```

Per onderdeel:

```
{
  naam,
  x_min, x_max,
  y_min, y_max,
  hoogte,
  frame_parent,
  waterverbinding,
  toegangsrichting
}
```

Exact berekenbaar:

```
lengte-breedteverhoudingen
oppervlakteverhoudingen
afstand tussen drempels
afname per omheining
hoogtetoename per niveau
torenpositie t.o.v. centrum
wateroppervlak vs. landoppervlak
routeafstand vs. rechte afstand
```

## NPR-Model Samenvatting

```
ANGKOR_WAT {
  SOURCE:    landschap + wateraanvoer
  FRAME_0:   gracht
  FRAME_1:   buitenmuur
  ROUTE:     westelijke causeway
  FRAME_2:   buitenhof en galerijen
  FRAME_3:   verhoogde tempelterrassen
  POINT:     centrale toren
  RETURN:    waterafvoer + terugkeer naar buitenveld
}
```

Kernverhouding = geen enkel magisch getal, maar herhaalde bewerking:

```
begrens → centreer → verklein → verhoog → herhaal
```

Wiskundig:

```
Fₙ₊₁ = schaal(Fₙ) + hoogte + strengere toegang
```
