# Kwelwater — Ondergrondse Return

## Definitie

```
regen valt op hogere zandgronden
→ zakt in de bodem
→ stroomt langzaam door zandlagen
→ komt in lager gelegen beekdalen omhoog
= kwelwater
```

Tientallen tot honderden jaren onderweg. Bevat mineralen (ijzer, kalk).

## Terminologie

```
kwel         = het proces (grondwater komt omhoog)
bron         = de plek waar het zichtbaar uittreedt
kwelbron     = zichtbare uitstroompunt
kwelgebied   = verspreide uitstroom (nat grasland, beekdal)
```

---

## NPR-OS Lezing

Kwel is de **ondergrondse return van regenwater naar het landschap**:

```
SOURCE    → regen op hoge zandgronden
FRAME     → zandlagen, ondergrondse route
CONTENT   → zoetwater stromend door de tijd
CONTENT-D → kwel, beekdal, vegetatie, bewoning, taalroute
```

Of in NPR-cyclus:

```
NOISE   → regen valt op hoogland
PATTERN → infiltratie, ondergrondse stroom (jaren/eeuwen)
RETURN  → kwel in beekdal (terug naar zichtbaar landschap)
```

---

## Volledige Route

```
hoge zandgrond
→ infiltratie
→ ondergrondse zoetwaterroute
→ kwel
→ beekdal
→ vegetatie
→ bewoning en taalroute
```

---

## Voorbeeld: Drenthe

- Drents Plateau (hoge zandgrond) → infiltratie → Drentsche Aa beekdalen
- Voedt: beken, blauwgraslanden, kalkmoerassen, bijzondere vegetatie
- Nationaal Park Drentsche Aa

---

## Relatie tot Overige Waterroutes

```
oppervlaktewater  → zichtbaar stroomt (rivier, kanaal, sloot)
ondergrondwater   → onzichtbaar stroomt (kwel, bron, grondwater)
```

Beide samen vormen het complete water-systeem:

```
REGEN
  ↓
INFRATIE (hoogland)
  ↓
GRONDWATER (ondergronds, tientallen-honderden jaren)
  ↓
KWEL (beekdal, laagland)
  ↓
BEKEN → RIJVEREN → BOEZEM → ZEE
```

---

## Juridische Relevatie

Kwelgebieden zijn vaak ecologisch waardevol en juridisch beschermd (Natura 2000, natuurgebieden). Historisch bepaalden ze:
- waar beekdalen ontstonden
- welke vegetatie mogelijk was
- waar bewoning zich vestigde
- welke taalroute volgde
