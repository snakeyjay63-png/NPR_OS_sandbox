# Kwelder — Levende Grens

## Positie in de Volledige Route

```
zee → wad → kwelder → dijk → boezem → polder
```

Kwelders zijn geen harde infrastructuur, maar een **levende overgangsbuffer** tussen zee en land.

---

## De Kwelder als Natuurlijke Watermachine

```
getij brengt water + sediment
→ planten remmen stroming
→ sediment bezinkt
→ bodem groeit
→ nieuwe vegetatie vestigt zich
→ golfkracht neemt af
```

Werkt tegelijk als:
- sedimentvanger
- golfdemper
- zout-waterbuffer
- groeizone van nieuw land
- leefgebied

---

## Vergelijking: Angkor vs. Nederland

```
ANGKOR
baray + kanaal + dijk + veld
→ kunstmatige bewaring

NEDERLAND
wad + kwelder + dijk + polder
→ natuurlijke voorverdediging + harde infrastructuur
```

**Baray** bewaart water binnen een kunstmatig frame.
**Kwelder** bouwt het frame zelf op door `plant × water × sediment × tijd`.

---

## Planten als Actieve Component

| Laag | Overstroming | Vegetatie |
|---|---|---|
| Lage kwelder | Vaak | Pionierplanten (zeekraal, Engels slijkgras) |
| Hogere kwelder | Minder vaak | Dichtere vegetatie (lamsoor, zeeaster, zoutmelde) |

Geen decoratie — planten veranderen stroming en houden slib vast.

```
geen vaste lijn land | water
maar:
water ↔ sediment ↔ plant ↔ land
```

---

## NPR-OS Weergave

```
SOURCE    → getij en sediment
FRAME     → kwelderoppervlak en kreken
CONTENT   → planten die stroming vertragen
CONTENT-D → nieuw land, kustbescherming, leefgebied
```

---

## Drie Grenstypen

```
DIJK    = harde grens (kunstmatige begrenzing)
KWELDER = levende grens (natuurlijke buffer)
POLDER  = beheerst binnenveld (gecontroleerd land)
```
