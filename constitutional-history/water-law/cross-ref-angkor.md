# Angkor ↔ Nederland — Hydraulische Landschappen

## Kern

Angkor is een **stad in een waternetwerk**.
Nederland is een **waternetwerk waarin steden en landvakken zijn geplaatst**.

Onafhankelijk ontstaan, vergelijkbare systemische vorm.

## Schaal

Angkor Wat zelf ≈ 2 km². UNESCO-park ≈ 400 km². Greater Angkor ≈ 1.000–3.000 km².
Nederland ≈ 41.543 km².

Angkor is **niet** zo groot als Nederland — de overeenkomst zit in de **schaalrelatie**:
```
Angkor Wat : Greater Angkor ≈ lokaal centrum : regionaal waterlandschap
Nederlandse stad : Nederland ≈ lokaal centrum : nationale delta
```

Zie `angkor-wat-nested-structure.md` voor het architecturale model.
Zie `angkor-thermodynamic-system.md` voor het thermodynamische model.

## Tegenvorm: Vasthouden vs. Wegpompen

| | Angkor | Nederland |
|---|---|---|
| **Klimaat** | Moesson (nat/droog seizoen) | Rivier + zee + regen (laag land) |
| **Hoofdopgave** | Extreme seizoenen beheersen | Water buiten houden + afvoeren |
| **Principe** | Water verzamelen om te voeden | Water verwijderen om droog te houden |
| **Involutie** | Droogmakerij = inverse van baray | Baray = inverse van droogmakerij |
| **Natuurlijke buffer** | — | **Kwelder** (levende golfdemper) |

### Beemster (1612)
```
BARAY:   begrensd vlak → gevuld met water
DROOGMAKERIJ: begrensd vlak → water eruit → land ontstaat
```

Beide: geometrisch raster van kaden, kanalen, verdeelde percelen.

## Hybride Landschap

Geen volledig kunstmatige leiding buiten de natuur. Koppeling aan:
- hoogteverschillen
- bodem
- rivieren
- seizoenen
- vegetatie
- sedimentstromen

```
natuur + infrastructuur + bestuur + onderhoud = levend watersysteem
```

Wanneer onderhoud stopt → netwerk verandert: sediment, blokkades, vegetatie, ongelijkheid.
