# Angkor — Thermodynamisch Water–Warmte–Landschapssysteem

## Kerninzicht

Angkor thermodynamisch lezen als **gekoppeld water–warmte–landschapssysteem**. Niet omdat de Khmer moderne vergelijkingen gebruikten, maar omdat hun ontwerp warmte, vocht, straling, stroming en opslag fysiek organiseerde.

---

## 1. Energiebalans

Vereenvoudigd voor een landschap:

```
inkomende zonnestraling
= opwarming bodem en gebouwen
+ opwarming lucht
+ verdamping
+ tijdelijke warmteopslag
```

Formeel:

```
Rₙ = H + LE + G + ΔS

Rₙ = netto stralingsenergie
H  = voelbare warmte naar lucht
LE = latente warmte voor verdamping
G  = warmte naar bodem en steen
ΔS = opgeslagen warmte
```

Angkor: wateroppervlakken, grachten, kanalen, vijvers, bomen, vochtige bodem, zware stenen constructies → zonne-energie verdeeld over meerdere uitgangen.

**Nuance:** minder boomkroonbedekking (o.a. Angkor Wat zelf) = hogere oppervlaktetemperaturen dan beboste terreinen. (Nature)

---

## 2. Water Zet Warmte Om In Verdamping

```
zonnewarmte → water verdampt → energie wordt latente warmte
→ lokale opwarming wordt geremd
```

```
Q = m × Lᵥ
```

Bij kamertemperatuur: 1 kg water ≈ `2,4 MJ` verdampingsenergie. Onttrekt energie aan water, bodem, lucht.

```
waterreservoir + natte bodem + plant + zon
→ evapotranspiratie
→ vochtige, relatief koelere lucht
```

**Nuance:** in al zeer vochtig tropisch klimaat werkt verdampingskoeling minder sterk dan in droge lucht. Schaduw, vegetatie, luchtbeweging, warmteopslag minstens zo belangrijk.

---

## 3. Bomen Zijn Warmtewisselaars

```
bladkroon    → onderschept zonnestraling
wortels      → nemen bodemwater op
huidmondjes  → verdampen water
stam + kroon → veranderen luchtstroming
```

Vegetatie = actief onderdeel energiebalans, niet decoratie.

Verlies bosbedekking → microklimaat verandert + zandsteenverwerking versnelt. (ScienceDirect)

Nederlandse vertaling:

```
waterberging + diepwortelende bomen + natte vegetatie + open luchtcorridor
= koeling + waterbuffering
```

Bomen verbinden met daadwerkelijk beschikbare waterreserve.

---

## 4. Steen Als Thermische Buffer

```
overdag:   zon → steen slaat warmte op
avond/nacht: steen → geeft warmte terug
```

```
Q = m × c × ΔT
```

Zware constructie warmt langzamer op, maar geeft 's nachts ook warmte af. Zonder schaduw en nachtelijke ventilatie = langdurige hitte.

Goede combinatie:

```
massa + schaduw + ventilatie + nachtelijke afkoeling
```

Nederland: zware gebouwen + buitenzonwering + koele groene binnenhoven + nachtventilatie.

---

## 5. Grachten Als Thermische Ring

```
warm landschap → watergrens → vochtige overgang → begrensd binnenveld
```

Thermische ring kan:
- zonnewarmte opslaan
- verdamping mogelijk maken
- dag-nachtritme vertragen
- vegetatie ondersteunen
- lokale luchtstroming beïnvloeden

Maar: stilstaand water ≠ automatisch koeler. Hangt af van watertemperatuur, diepte, wind, luchtvochtigheid, beschaduwing, verhouding wateroppervlak/bebouwing.

Nederlands principe:

```
water + schaduw + luchtstroming + vegetatie + doorstroming
= bruikbaar microklimaatsysteem
```

Niet: `meer open water = altijd koeler`

---

## 6. Hoogteverschil Als Opgeslagen Energie

Angkor: natuurlijk verval tussen hoger terrein → stedelijk gebied → Tonlé Sap-bekken.

```
Eₚ = mgh

hoog waterpeil → kanaal → lager veld → distributie zonder pomp
```

Nederland werkt vaak omgekeerd:

```
lage polder → gemaal → hogere boezem
E_minimaal = ρ × g × V × Δh
```

Voor 1 m³ water 1 meter omhoog: ≈ 9,81 kJ (zonder pompverliezen).

**Les:** zwaartekracht en tijdelijke opslag eerst, pomp pas wanneer systeemverschil vereist.

```
regen op dak → vasthouden → lokaal gebruiken → vertraagd infiltreren
→ alleen overschot afvoeren
```

In plaats van:

```
regen → direct riool → gemaal → snel buiten gebied
```

---

## 7. Seizoensbuffer Als Thermodynamische Batterij

Baray slaat niet warmte op als elektrische batterij, maar:

```
massa + potentiële energie + vochtbeschikbaarheid + tijd
```

Natte periode:
```
grote waterinstroom → tijdelijke opslag → minder piekafvoer
```

Droge periode:
```
opgeslagen water → irrigatie → bodemvocht → evapotranspiratie
→ voedselproductie en koeling
```

Nederland (wateroverlast + droogte):

```
winteroverschot → regionaal vasthouden
zomerbehoefte   → lokaal beschikbaar maken
```

---

## 8. Van Afvoersysteem Naar Circulerend Watersysteem

Klassiek Nederland:
```
wateroverschot → zo snel mogelijk weg
```

Angkor-achtig:
```
vang → verdeel → berg → gebruik → infiltreer → voer alleen overschot af
```

### Schalen

**Gebouw:**
```
dak → blauw-groen dak → opslag → planten → verdampingskoeling → hergebruik
```

**Straat:**
```
regen → verlaagde groenstrook → wadi → boomwortelzone → infiltratie
```

**Wijk:**
```
waterplein → tijdelijke piekberging → vertraagde afvoer → openbare ruimte
```

Rotterdamse waterpleinen: zware regen = tijdelijke berging → daarna gecontroleerd leeg → normaal = openbare ruimte.

**Regio:**
```
rivier → uiterwaard → nevengeul → nat natuurgebied → grondwateraanvulling
→ gecontroleerde afvoer
```

## 9. Geneste Thermische Velden

Sterkste overeenkomst wanneer onderdelen **genest** worden:

```
woning
binnen
  groen-blauw bouwblok
binnen
  waterbergende wijk
binnen
    polder
binnen
      boezem
binnen
        rivierdelta
```

Elke laag:

```
CAPTURE → STORE → DISTRIBUTE → COOL → INFILTRATE → RETURN
```

```
dak    → vangt millimeters
straat → vangt centimeters
plein  → vangt buien
park   → vangt wijkwater
polder → vangt seizoenswater
rivier → vangt regionale pieken
```

## 10. Kwelders En Veen — Nederlandse Thermische Tegenvormen

### Kwelder

```
getij → vegetatie remt stroming → sediment bezinkt
→ golfenergie gedissipeerd → land groeit
```

Inkomende bewegingsenergie verdeeld over:

```
wrijving + turbulentie + plantbuiging + sedimentatie
```

Niet hard teruggekaatst, maar verdeeld en gedissipeerd.

### Nat Veen

```
hoog waterpeil → minder oxidatie → minder bodemdaling
→ meer verdamping → minder extreme temperaturen
```

Nuance: waterkwaliteit en methaan meewegen. "Natter" ≠ universele oplossing. Peil en landgebruik per bodemtype berekenen.

---

## 11. Kwetsbaarheid Van Angkor — De Belangrijkste Les

Angkors netwerk groeide complexer. Uitzonderlijke droogte + extreme neerslag wisselend = strategische knooppunten kwetsbaar → schade verspreidt door netwerk. (PMC)

Thermodynamisch én netwerkmatig:

```
maximale verbinding ≠ maximale veerkracht
```

Robuust systeem heeft:

```
meerdere routes
+ lokale opslag
+ afsluitbare compartimenten
+ gecontroleerde overloop
+ herstelbare knooppunten
```

Niet één nationale watermachine die altijd perfect moet functioneren:

```
zelfstandige lokale cellen ↔ gekoppeld wanneer nodig
```

## Nederlands Angkor-Principe

```
NEDERLAND_THERMAL_WATER_FIELD {

  SOURCE:
    zon + regen + rivier + kwel + getij

  CAPTURE:
    dak + bodem + park + uiterwaard

  STORAGE:
    blauw-groen dak + bassin + boezem + grondwater

  CONVERSION:
    verdamping + vegetatie + thermische massa

  ROUTING:
    zwaartekracht waar mogelijk
    pomp waar noodzakelijk

  PROTECTION:
    dijk + kwelder + overstroombaar landschap

  RETURN:
    infiltratie + hergebruik + gecontroleerde afvoer
}
```

## Kern

```
Angkor:      water ordent een warm seizoenslandschap
Nederland:   water kan een nat én opwarmend deltalandschap ordenen
```

Toepasbare thermodynamische regel:

```
voer energie en water NIET direct af

maar:

vang → vertraag → verdeel → zet om → hergebruik → laat gecontroleerd terugkeren
```

Resultaat: minder piekbelasting, meer droogtereserve, verdampingskoeling, gezondere vegetatie, systeem dat bij uitval niet als geheel faalt.
