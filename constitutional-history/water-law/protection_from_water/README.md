# Verdediging tégen Water

Water als bedreiging. Collectief beheer als oplossing.

## Principe

```
dijkplicht + schouw + keur + omslag = blijvende droogte
```

## Waterschap als Functionele Rechtsorde

- **Regel stellen:** keur
- **Uitvoeren:** dijk, sluis, gemaal onderhouden
- **Belasting heffen:** omslag/waterschapsbelasting
- **Controleren:** schouw
- **Handhaven:** herstelbevel of sanctie
- **Geschillen:** eigen procedure

## Netwerkstructuur

```
perceel → polder → waterschap → hoogheemraadschap → gewest → staat
```
