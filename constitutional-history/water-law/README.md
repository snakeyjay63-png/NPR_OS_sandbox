# Waterrecht — Tegenstander, Infrastructuur, Wapen

## Twee Standen van Één Infrastructuur

```
NORMALE STAND
water afvoeren → land droog en bruikbaar

DEFENSIEVE STAND
water aanvoeren en vasthouden → land onbruikbaar (inundatie)
```

Dezelfde dijken, sluizen, kanalen, gemalen, inlaten en overlaten.

## NPR-OS Lezing

### Protection from Water
```
Noise:   gedeeld fysiek probleem (water als bedreiging)
Pattern: keur + schouw + omslag + bestuur
Return:  blijvend droog en bruikbaar land
```

### Defence with Water
```
Noise:   militair gevaar (vijand)
Pattern: inundatie + forten + inlaten
Return:  beschermd land (via tijdelijk onbruikbaar maken)
```

---

## Twee Bestuurstypen

```
water als bedreiging
→ collectief civiel bestuur (waterschap)

water als wapen
→ gecentraliseerd militair bestuur (Ridderschap, Staten-Generaal)
```
