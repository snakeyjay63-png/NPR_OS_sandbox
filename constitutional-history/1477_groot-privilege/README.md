# Groot Privilege — 1477

**Datum:** 11 februari 1477
**Uitgegeven door:** Marianne van Bourgondië
**Type:** Privilege

## Betekenis
Beperking van vorstelijke macht. Eerste systematische uitspraak dat de landsheer gebonden is aan bestaande rechten en instemming van de gewesten.

## Beginselen
- `limited_government` — vorst ≠ onbeperkte macht
- `fundamental_rights` — bescherming bestaande privileges
- `consent_to_taxes` — geen belasting zonder instemming
