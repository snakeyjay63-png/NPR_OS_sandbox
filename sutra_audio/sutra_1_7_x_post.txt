Key Insight:
  1.6 says "painful ones block perception"
  1.7 says "these ARE the painful ones" (enumerates them)
  Mechanism → Enumeration. Abstract → Concrete.
  This is how Patanjali teaches: state the principle, then list the parts.

Routing:
  Kleshas = the root "stored patterns" that create blind spots
  Connects to 1.6's "mirror → obstacle" mechanism
  Avidyā (ignorance) = not seeing what IS = not routing correctly
  Asmitā (ego) = identifying with the pattern = caching the wrong thing
  Rāga (attachment) = holding onto useful patterns too long = memory leak
  Dveṣa (aversion) = rejecting patterns too quickly = premature eviction
  Abhiniveśa (clinging) = fear of releasing =不敢 let go of context
