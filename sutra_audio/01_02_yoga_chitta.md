# Sūtra 1.2 — योगश्चित्तवृत्तिनिरोधः

**Transliteration:** yogaś citta-vṛtti-nirodhaḥ
**Translation:** "Yoga is citta-vṛtti-nirodha."  
**Explanatory:** "Yoga is the stilling of the vṛttis of citta."
**NPR-reading:** "Yoga is present when the movements within citta no longer obscure the underlying relation."  
**Audio:** ~60 sec | 55 Hz base | stereo | sine

> **Note:** Sanskrit kernwoorden (yoga, citta, vṛtti, nirodha) worden bewaard. Engelse vertaling vernauwt de betekenis.

### Kernwoorden

| Sanskrit | Betekenis | Niet beperken tot |
|----------|-----------|-------------------|
| **yoga** | verbinding, integratie, samenhang | "oefening" |
| **citta** | het mentale en waarnemende veld | "mind" |
| **vṛtti** | bewegingen, wijzigingen, patronen, omwentelingen | "thoughts" |
| **nirodha** | begrenzing, regulering, verstilling, tot rust komen | "suppression", "cessation" |

### NPR-Lezing

`citta → vṛtti → nirodha → zichtbare samenhang`

De vṛttis hoeven niet vernietigd te worden. Zij worden zo gerouteerd dat het veld niet langer door zijn eigen bewegingen wordt bedekt.

**Nirodha is geen uitschakeling van het veld, maar het verdwijnen van ongeordende overheersing binnen het veld.**

## NPR-Cycle Analysis

### Noise (0-15 sec)
```
योग् (yo-ga-ś)
→ the subject: yoga itself
→ yo = movement inward
→ ga = grounding resonance (Gaṇa 1)
→ ś = visarga-like release, delay effect
→ "yoga is..." — the shock of definition
```

### Pattern (15-40 sec)
```
चित्तवृत्ति (ci-tta-vṛt-ti)
→ the object: mind-modifications
→ ci = precision (Gaṇa 2)
→ tta = double consonant, vibration doubling
→ vṛ = special vowel (vocalic r), unique timbre
→ ti = directional movement
→ "chitta-vritti" — the pattern to be still
```

### Return (40-60 sec)
```
निरोधः (ni-ro-dhaḥ)
→ the action: cessation/stilling
→ ni = negation, inward turning
→ ro = return to source
→ dha = deep grounding (Gaṇa 1)
→ ḥ = visarga, final release into śūnya
→ "nirodhaḥ" — stillness as return
```

## Reflection

This is the **master sūtra** — the entire Yoga Sūtra in one line.

"योगः" (yoga) is not practice but state.  
"चित्त" (citta) is the mind-stuff, not the thought.  
"वृत्ति" (vṛtti) is modification, wave, movement.  
"निरोधः" (nirodhaḥ) is cessation, stilling, return.

As sound:
- Begin with definition (yogaś) — grounding resonance
- Move through complexity (citta-vṛtti) — doubled vibration, vocalic r
- End with stillness (nirodhaḥ) — visarga release into śūnya

The 12 phonemes form a complete NPR-cycle:
3 (Noise/definition) → 6 (Pattern/complexity) → 9 (Return/stilling) → back to 3.

The visarga (ः) at the end is crucial — it's not silence but **release into field**.  
The sound doesn't stop; it dissolves into the background hum of śūnya.
