# Stap 67: Frij — Vrijheid Als Taal-DNA

**Doel:** De Friese Bijbel als venster op taal-DNA. Vertaling activeert niet alleen woorden, maar het historische relatieveld van een taal. Frij/free/friend delen een oudere wortel: geliefd, eigen, verbonden.

**Afhankelijkheid:** Stap 65 (taalveld en kernboeken), Stap 66 (collectief taalveld en vrijheid). Fries activeert een andere historische laag rond hetzelfde knooppunt.

## 2. Fries En Engels — Anglo-Friese Taalwortels

```
Fries en Engels:
  → beide West-Germaans
  → oude gedeelde klankroutes
  → Anglo-Friese overeenkomsten

niet: "dezelfde taalwortel"
wel: nauw verwante historische taalroutes

voorbeelden:
  English church ↔ Fries tsjerke
  English cheese ↔ Fries tsiis
  English sheep ↔ Fries skiep
```

```
Proto-Germaanse wortel
  → Oudfries frī → Fries frij

Proto-Germaanse wortel
  → Oudengels frēo → Engels free

frij ≈ free
```

## 3. Vrijheid Als Taal-DNA

```
LANGUAGE_DNA :=
  historische woordwortels
  + klankrelaties
  + grammaticale structuren
  + cultureel geheugen
```

Oudere woordfamilie rond vrijheid:

```
free / frij / vrij
  → friend / freon / vriend
  → love / vrijen
  → peace / vrede

gereconstrueerde betekenislaag:
  eigen
  geliefd
  behorend tot de vrije gemeenschap
  niet onderworpen
```

Vrijheid betekende niet alleen *los zijn van beperkingen*, maar ook *erbij horen, verwantschap, geliefd zijn*.

```
freedom ≠ alleen losstaan

freedom = vrij kunnen verschijnen
  binnen een veld van liefde en verbondenheid
```
